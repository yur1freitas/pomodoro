# Pomodoro

[![License][LICENSE_BADGE]][LICENSE_BADGE]
[![Node.js Version][NODE_BADGE]][NODE_BADGE]
[![PNPM Version][PNPM_BADGE]][PNPM_BADGE]

Um simples site de Pomodoro que permite o usuário iniciar sessões de trabalho de `25min` intercaladas com pausas de `5min`, onde o término de cada sessão é alertado por um efeito sonoro. Além disso, é possível pausar, pular e resetar sessões
